# Mirrah Nashihin E-Portfolio

Static multi-page e-portfolio website for GitHub Pages.

## Pages

- `index.html` - homepage
- `about.html` - profile and tagline options
- `interest.html` - SOC Analyst interest
- `skills.html` - technical and soft skills
- `projects.html` - FYP and cybersecurity project evidence
- `activities.html` - leadership and extracurricular experience
- `certificates.html` - certificates and project evidence
- `career.html` - target companies and career plan
- `reflection.html` - personal reflection based on company and career questions
- `contact.html` - contact details and resume placeholder

## How to Upload to GitHub

1. Create a new GitHub repository.
2. Upload all files from this folder.
3. Go to repository `Settings`.
4. Open `Pages`.
5. Select branch `main` and folder `/root`.
6. Save and wait for GitHub Pages to publish the website.

## Certificate Images

For certificate thumbnails, use PNG or JPG files. Keep the original PDF if needed, but images are easier to preview in the website.

Suggested folder:

```text
assets/certificates/
```

Then replace the placeholder area in `certificates.html` with an image:

```html
<img src="assets/certificates/certificate-name.png" alt="Certificate name">
```
